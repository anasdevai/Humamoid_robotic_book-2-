# Chapter 24: Isaac Sim Setup and Configuration

**Week 8 | Module 3 | Time: ~4 hours**

## Learning Objectives
- Install and set up NVIDIA Isaac Sim.
- Understand the Omniverse platform and its connection to Isaac Sim.
- Configure basic Isaac Sim environments and project settings.

## Prerequisites
- Chapter 23: NVIDIA Isaac Platform Overview.
- NVIDIA GPU (RTX series recommended).

## Overview
This chapter provides a practical guide to installing and configuring NVIDIA Isaac Sim, the powerful robotics simulation platform built on NVIDIA Omniverse. We will walk through the setup process, introduce the Omniverse ecosystem and its integration with Isaac Sim, and cover essential steps for configuring basic simulation environments and project settings, preparing you for high-fidelity robotics development.

## Key Concepts
- Isaac Sim installation
- NVIDIA Omniverse platform
- Isaac Sim environment configuration
- Project settings and asset management

## What You'll Build
- A configured Isaac Sim environment.
- Launch a basic robot in a sample world.
